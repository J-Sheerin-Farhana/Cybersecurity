# Cybersecurity Learning Repository
This repository documents my cybersecurity learning journey. It contains structured theory notes, hands-on labs, cybersecurity tools, and certification preparation materials.

The goal of this repository is to build strong practical and theoretical knowledge in cybersecurity while maintaining organized documentation of concepts, experiments, and lab exercises.

## Repository Structure

### Theory Lessons
Contains detailed notes on core cybersecurity concepts such as:
- Networking fundamentals
- Linux security
- Cryptography
- Web security
- Malware analysis
- Security operations (SOC)

### Hands-On Labs
Practical cybersecurity exercises and experiments including:
- Kali Linux labs
- Nmap scanning practice
- Wireshark packet analysis
- Metasploit exploitation
- Capture The Flag (CTF) exercises

### Courses and Certifications
Documentation and study notes for cybersecurity certifications and training programs.

### Tools
Guides and usage notes for commonly used cybersecurity tools such as:
- Nmap
- Wireshark
- Burp Suite
- Metasploit

## Purpose

This repository serves as a personal knowledge base and portfolio to demonstrate continuous learning and practical experience in cybersecurity.
